# NNN: 작업 이름

## 사람용 (AI에게 보내지 않는 부분)

- 작성자: HY 또는 JK
- 날짜: YYYY-MM-DD
- ROADMAP 항목: (예: 확장 1단계)
- 커밋 메시지: `[HY/expand] ...` 또는 `[JK/refactor] ...`

사전 준비:
- [ ] git pull
- [ ] (확장 작업이면) 기존 보고서 HTML 다운로드 백업

검증 체크리스트 (ROADMAP.md의 해당 단계에서 복사):
- [ ] npm run build 통과
- [ ] ...

## AI에게 보낸 프롬프트 원문

설계 방식이나 파일 구조를 지시하지 않는다.)
